# release_checklist.md - Checklist automatizado de release

- [ ] Todos os testes unitários, E2E e integração passando
- [ ] Lint (pylint) sem erros críticos
- [ ] Métricas de performance e uso revisadas (ver /metrics)
- [ ] Logs de erro e latência revisados
- [ ] Checklist de acessibilidade validado
- [ ] Documentação e runbook atualizados
- [ ] Roadmap e ADRs revisados
- [ ] Feedback de usuários incorporado (UX)

> Atualize este checklist a cada release.
